# ThermoriaVerse
[English](./README.md)

**Gra sandbox we wczesnej fazie rozwoju, skupiona na eksploracji nieskończonego świata, progresji postaci i osiągnięciach, tworzona w C#.**

## Oficjalny Opis gry
Jako Architekt, podróżuj po planecie stworzonej z voxeli — pełnej barwnych krajobrazów, życia i ukrytych potężnych sił.
Kształtuj światy według własnych pomysłów: podnosząc góry, obniżając krainy lub tworząc olbrzymie budowle. Każdy voxel, każda roślina i każda istota reagują na Twoją obecność.
Buduj i niszcz w duchu klasycznych gier sandbox, połączonych z nowoczesnym podejściem 3D w stylu voxelowego pixel artu.
Odkrywaj unikatowe biomy, starożytne ruiny, ukryte sanktuaria oraz miasta tętniące życiem.
Przemierzaj świat pieszo lub obserwuj go z góry — jako wszechwiedzący Architekt.
Steruj pojedynczymi NPC lub obserwuj z góry. Pomagaj im przetrwać, rozwijać się, walczyć lub budować cywilizacje, lub pozostaw ich tak, jak są i oglądaj swój świat.
Każda decyzja niesie konsekwencje — możesz stać się ich wybawcą albo niszczycielem.
Proceduralne generowanie świata, biomy i roślinność, dynamiczny system chunków oraz optymalizacja GPU, to wszystko sprawia, że gra działa płynnie nawet w ogromnych, złożonych krainach, z dużym zasięgiem renderu.
**ThermoriaVerse** — twórz własne multiversum.

## Wizja Gry
Podstawowa koncepcja rozgrywki i postęp prac są już ustalone.
Obecnie rozwój skupia się na tworzeniu zaplanowanych doświadczeń dla graczy.

## Wymagania systemowe

### Minimalne wymagania

- System Operacyjny: **Windows 10 - 64 bit / Windows 11 - 64 bit**
- Procesor: **Intel Core 2 Duo E4400 / AMD E2-6110**
- Pamięć: **4 GB RAM**
- Karta Graficzna: **NVIDIA GeForce 9300M / AMD Radeon 530**
- Sieć: **Stałe szerokopasmowe połączenie z Internetem**
- Pamięć: **5 GB dostępnej przestrzeni**
- Karta dźwiękowa: **kompatybilna z DirectX**

### Zalecane wymagania

- System Operacyjny: **Windows 11 - 64 bit**
- Procesor: **Intel Core i3-8100 / AMD Ryzen 3 2200G**
- Pamięć: **8 GB RAM**
- Karta Graficzna: **NVIDIA GeForce GTX 650 2GB / AMD Radeon R7 2GB**
- Sieć: **Stałe szerokopasmowe połączenie z Internetem**
- Pamięć: **8 GB dostępnej przestrzeni**
- Karta dźwiękowa: **kompatybilna z DirectX**
